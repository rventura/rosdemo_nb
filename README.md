# Simple ROS demos in Jupyter notebook format

This repository features two Jupyter notebooks featuring topic
communication using the ROS framework, focused on simplicity.

The code was largely inspired in the standard ROS tutorials available
here:
https://wiki.ros.org/ROS/Tutorials/WritingPublisherSubscriber%28python%29

This material targets the Autonomous Systems course at Instituto
Superior Técnico, University of Lisbon.
